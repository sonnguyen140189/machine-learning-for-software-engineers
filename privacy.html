Privacy Policy — Phu Quoc Tips

Last updated: 2026-05-18

This automated content-distribution bot does not collect, store, or process any personal data from any user.

1. Data Collection

The Service has no user-facing surface. There are no accounts, no logins, no forms, no comments, no analytics SDKs, and no cookies. The Service only writes to its own Facebook Page, Instagram Business account, and TikTok account using server-to-server APIs.

2. Third-Party APIs

The Service makes outbound calls to:

  • Google Places API — to fetch publicly available place details for Phu Quoc.
  • Pexels and Unsplash APIs — to fetch supplementary public-license photography.
  • Anthropic Claude API — to generate post captions from place metadata.
  • Meta Graph API and TikTok Content Posting API — to publish the bot's own posts.

None of these calls transmit data about any third party. They only request public reference data or publish content owned and authored by the operator.

3. Cookies

The Service has no web frontend and sets no cookies.

4. Data Retention

The Service retains only its own state (which places it has already posted about) so it doesn't repeat. No user data is retained because no user data is collected.

5. Children

The Service is not directed at children under 13 and does not knowingly process their data (none is collected from anyone).
