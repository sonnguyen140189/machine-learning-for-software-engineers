# Background music for slideshow videos

Drop royalty-free / CC0 audio tracks here (`.mp3`, `.m4a`, `.wav`, `.aac`, `.ogg`).
The bot picks one at random per video, loops it to the slideshow length,
applies a 1-second fade in/out, and mixes it at 60% volume so the on-screen
captions still pop.

If this directory is empty, video posts ship silent — the build does not crash.

## Where to source tracks

All of these allow commercial use and don't require attribution for the bot's
use case (auto-generated travel content). Pick "tropical", "summer", "beach",
"holiday", "vlog", or "uplifting" vibes.

- **Pixabay Music** — https://pixabay.com/music/search/tropical/  (download MP3, CC0)
- **YouTube Audio Library** — https://studio.youtube.com (Audio Library tab, filter by Genre = Pop / Reggae / Cinematic)
- **Mixkit** — https://mixkit.co/free-stock-music/tag/tropical/  (free, no attribution required)
- **Bensound** — https://www.bensound.com/royalty-free-music/tropical  (free tier requires attribution; skip unless you want to credit)

Aim for 5–10 tracks so the same one doesn't repeat too quickly across daily
posts. Each track 30–90 seconds is plenty — the slideshow is typically
18–21 seconds and the audio is trimmed to match.
